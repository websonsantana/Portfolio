Sisteminha de controle em Excel

Projeto em Excel para empresa de frios com dashboards interativos para análise de dados e controle de indicadores.

Este projeto em Excel foi desenvolvido para uma empresa de frios, com o objetivo de identificar perdas na produção. Utilizando dashboards interativos, painéis dinâmicos e ferramentas avançadas do Excel, a solução facilita a análise de dados, permitindo um controle mais preciso dos indicadores de desempenho. Com essa ferramenta, a empresa pode otimizar processos, reduzir desperdícios e tomar decisões mais assertivas, garantindo uma visão completa da produção.

ERP LOJA DE MÓVEIS

Sistema ERP completo para gestão de lojas de móveis com controle de clientes, custos e entregas.

O cliente necessitava de uma página que pudesse gerenciar suas lojas, banco de controle de clientes, custos das lojas e acompanhamento de entregas, com dashboards para tomada de decisões. O sistema desenvolvido oferece uma solução completa para gestão do negócio, com módulos específicos para cada área da empresa, integração entre setores e relatórios personalizados para análise de desempenho.

Sistema de Gestão

Sistema para gestão de metas de colaboradores com acompanhamento de habilidades e experiências.

Este projeto foi desenvolvido utilizando HTML, CSS e JavaScript. Desenvolvido para gerenciar metas de colaboradores dentro de empresas, onde gestores e colaboradores podem inserir metas, suas experiências e habilidades para facilitar a tomada de decisões no desenvolvimento da equipe. Utilizado pelos setores de Gestão e RH, o sistema permite o acompanhamento individual e coletivo do desempenho, identificação de gaps de competências e planejamento de treinamentos.

Projeto em 3D

Modelagem 3D de uma casa com SketchUp, incluindo detalhes arquitetônicos e disposição de móveis.

Este projeto foi desenvolvido utilizando a ferramenta SketchUp para modelagem tridimensional de uma casa. O modelo 3D da casa foi projetado com precisão para representar detalhes arquitetônicos e estruturais. Com o uso do SketchUp, foi possível criar um design realista, permitindo visualizações imersivas dos espaços internos e externos. O projeto inclui disposição de móveis, iluminação e acabamentos, auxiliando na concepção e planejamento da construção.
